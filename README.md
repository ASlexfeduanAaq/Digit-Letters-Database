# 数字字母数据库

这是一个包含从个位到万位所有数字字母的开源数据库。

## 数据结构

- `ones.txt`：包含 0-9 的字符表示
- `tens.txt`：包含 00-99 的字符表示
- `hundreds.txt`：包含 000-999 的字符表示
- `thousands.txt`：包含 0000-9999 的字符表示
- `ten_thousands.txt`：包含 00000-99999 的字符表示

## 使用方法

您可以直接下载这些文件，或者通过 Git 克隆该仓库来访问数据。
